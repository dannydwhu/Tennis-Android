## 常见问题

### Q1: 为什么识别准确率不是 100%？
A: 模拟数据包含随机噪声，且算法使用简化的规则引擎。实际测试中 85%+ 的准确率是可接受的。

### Q2: 如何提高特定类型的识别率？
A: 参见 `SwingClassifier.kt` 中的分类逻辑，调整对应阈值参数。

### Q3: 模拟数据和真实数据的差异？
A: 模拟数据基于理想模型生成，真实数据可能包含更多噪声和不规则性。建议在真机测试前先用模拟数据验证算法逻辑。
